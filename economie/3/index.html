1. Costurile de Producție (Lecția 2.5)

Totalitatea cheltuielilor pentru producerea, depozitarea și vânzarea bunurilor.

Elemente de bază:
CT = CF + CV
CTM = CT / Q | Cmg = ΔCT / ΔQ

Amortizarea (A): Recuperarea treptată a valorii capitalului fix (Kj).

A = Valoarea K fix / Durata de utilizare

  • Pământ (P): WMP = Q/P | WmgP = ΔQ/ΔP

3. Profitul (Lecția 2.7)

Formule cheie:
Pr Brut = IT - CT = (P - CTM) * Q
Pr Net = Pr Brut - Impozit
Rata Pr (la cost) = (Pr / CT) * 100

Pragul de rentabilitate: Q unde IT = CT (Pr = 0).

Producția Optimă: Inregistrată când diferența dintre Preț (P) și Cost Mediu (CTM) este maximă.
