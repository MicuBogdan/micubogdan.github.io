# Flashcarduri — Istoria Evreilor în România

Deschide `index.html` în browser (desktop sau mobil). Site-ul este responsive.

Comenzi rapide:

```bash
# deschide local (Linux)
xdg-open index.html
```

Dacă vrei, pot porni un server local simplu (ex: `python3 -m http.server`) pentru testare pe alte dispozitive din rețea.
